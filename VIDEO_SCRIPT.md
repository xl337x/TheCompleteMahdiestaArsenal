# 60-Second LinkedIn Video Script — Mahdiesta Arsenal

## Goal
Show the project as a clean, professional, browser-based offensive-security toolkit.

## Tone
Fast-paced, confident, no voiceover needed (use captions + music).

## Recording Specs
- Resolution: 1728×1117 (13" MacBook Pro aspect ratio) or 1920×1080
- Aspect ratio: 16:10 or 16:9
- Frame rate: 30fps
- Format: MP4

---

## Shot Breakdown

### 0:00–0:05 — Hook: Terminal intro
**Action:** Load `index.html` in Chrome, full-screen browser.
**Visual:** The typing terminal header `mahdiesta@arsenal:~#` animates.
**Caption:** "Tired of scattered cheat sheets?"
**Tip:** Wait for the typing carousel to cycle 2–3 commands.

### 0:05–0:12 — Reveal the arsenal
**Action:** Let the page finish loading. Slow scroll down slightly to show filter buttons and first card row.
**Visual:** Filter buttons (All 14, Active Directory 5, SQL Injection 2, Web 4, OSINT & Dorks 1, PrivEsc 2).
**Caption:** "Meet Mahdiesta Arsenal — 14 interactive offensive-security cheat sheets."

### 0:12–0:20 — Search capability
**Action:** Press `/` then type `mimikatz`.
**Visual:** Live search results appear as chips showing which sheets cover it.
**Caption:** "Search any tool, technique, or command — instantly."

### 0:20–0:30 — Category showcase
**Action:** Click filter buttons rapidly: AD → SQL → Web → PrivEsc.
**Visual:** Cards filter smoothly by category.
**Caption:** "Active Directory. SQLi. Web. PrivEsc. All in one place."

### 0:30–0:40 — Open a sheet
**Action:** Click "ADCS Attack Reference" card.
**Visual:** Sheet opens in new tab, show the command table and variables.
**Caption:** "Every sheet has copyable commands, variable auto-fill, and dark/light mode."
**Tip:** Press `t` to toggle light/dark theme quickly.

### 0:40–0:50 — Second sheet + theme toggle
**Action:** Switch to SQL Injection Operator or SQL Injection RCE tab.
**Visual:** Show the attack guide / payload generator.
**Caption:** "SQLi to RCE, web enumeration, auth bypass — ready for real ops."

### 0:50–0:55 — GitHub ribbon + CTA setup
**Action:** Back to index.html, mouse over the GitHub corner ribbon.
**Visual:** Show the ribbon and footer.
**Caption:** "Open source. Free. Deploy on GitHub Pages in one click."

### 0:55–1:00 — Outro
**Visual:** Freeze on the index header.
**Caption:** "Mahdiesta Arsenal — link in comments."
**Text overlay:** "Star ⭐ · Fork 🍴 · Share"
